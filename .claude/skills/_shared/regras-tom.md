# Regras de Tom — Referência Compartilhada

> Referência rápida para todas as skills que geram texto público.

## Regra principal

Seguir `_memoria/preferencias.md` estritamente. Se estiver vazio, usar os defaults abaixo.

## Defaults (quando preferencias.md estiver em branco)

- Frases curtas, parágrafos de 2-4 linhas
- Concreto: números, certificações, datas, valores quando souber
- Sem jargão de marketing/inglês quando o público não usa
- Sem corporativês ("alavancar", "sinergia", "vamos juntos!")
- Sem emojis decorativos em texto profissional (OK em legendas de rede social)
- Tom direto, humano, como se falasse com o vizinho

## O que NUNCA fazer (universal)

- Inventar dados, números ou estatísticas
- Usar superlativos não-comprovados ("o melhor", "número 1")
- Copiar respostas idênticas entre peças diferentes
- Prometer o que não depende só da empresa
